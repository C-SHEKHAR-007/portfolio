# Portfolio Variants

A collection of personal portfolio designs built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools.

## Variants

| File | Theme | Description |
|---|---|---|
| [`portfolio-glass.html`](./portfolio-glass.html) | Glassmorphism | Dark glassmorphism theme with parallax blobs, typewriter effect, devicons, and modal project cards |
| [`portfolio-minimalist.html`](./portfolio-minimalist.html) | Minimalist | Clean minimal layout with restrained typography and white space |
| [`portfolio-cinematic.html`](./portfolio-cinematic.html) | Cinematic | Scroll-driven "video playing" experience — film grain, letterbox bars, video timeline scrubber, scene-based navigation |
| [`terminal_look_portfolio/v7.html`](./terminal_look_portfolio/v7.html) | Terminal / Hacker | Dark hacker theme with particle canvas, custom cursor, tab-based experience panels, and section flash transitions |

## Tech

- Vanilla HTML5 / CSS3 / JavaScript (ES6+)
- [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans)
- [Devicons CDN](https://devicons.github.io/devicon/) for skill icons
